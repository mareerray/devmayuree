# DevMayuree Portfolio

A personal portfolio website showcasing my work, background, and tech stack as a web developer in training.

## Features

- Responsive layout with a modern, minimal design
- About section with a short professional introduction
- “What I Do” section describing focus areas
- Tech stack badges (Java, Spring Boot, Angular, React, JavaScript, TypeScript, Go, MongoDB, SQLite, GraphQL, HTML, CSS, Bootstrap, Postman, Docker, WebSocket)
- Project cards linking to individual project pages

## Tech Stack

- HTML 
- CSS
- JavaScript
- Bootstrap

## Project Structure

- `index.html` – main landing page
- `portfolio-item.html` – individual project pages
- `css/` – stylesheets
- `js/` – JavaScript files (interactive behaviour)
- `img/` – images and background assets
